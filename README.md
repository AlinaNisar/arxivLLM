Resume Intelligence Portal
Overview

Resume Intelligence Portal is an AI-powered recruitment prototype that helps recruiters efficiently screen, analyze, and shortlist candidates. The system processes uploaded resumes against a job description and provides AI-assisted recommendations to support interview selection.

Features
Upload a Job Description (PDF)
Upload multiple candidate resumes (PDF)
AI-powered resume parsing
Resume Library with uploaded files
Candidate rankings based on job fit
Resume analysis with extracted skills and experience
AI processing status for each resume
Search and filter candidates
Export rankings to CSV
AI-generated interview questions based on the uploaded Job Description
Recruiter dashboard with candidate insights
Dashboard Modules
Rankings: View AI-ranked candidates and recommendations.
Resume Library: Browse uploaded resumes and view candidate details.
Interview Questions: Generate and review interview questions with expected answers.
Technology Stack

Project Workflow
Upload a Job Description.
Upload one or more candidate resumes.
Execute Batch AI Parsing.
Review AI-generated candidate rankings.
Browse resumes in the Resume Library.
Generate interview questions from the Job Description.
Shortlist candidates for interviews.
Export the ranked candidate list as a CSV.
Project Goal

The purpose of this prototype is to demonstrate how AI can streamline the recruitment process by automating resume screening, improving candidate selection, and assisting recruiters with interview preparation.

Note: This is a prototype intended for demonstration and educational purposes. AI-generated recommendations should support—not replace—human hiring decisions.
